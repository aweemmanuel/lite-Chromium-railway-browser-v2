🔑 IMPORTANT SETTINGS
================================================================================

PERSISTENT STORAGE (CRITICAL!)
  Without: Bookmarks and downloads lost on every restart
  With: Everything survives forever

  Setup in Railway:
    1. Your project → Volumes
    2. New Volume → Mount /data
    3. Redeploy

DEFAULT WEBSITE
  Change what page opens:
    railway env add HOMEPAGE https://example.com
    railway redeploy

SCREEN RESOLUTION
  Adjust for performance/quality:
    railway env add WIDTH 1280    # Smaller
    railway env add HEIGHT 720
    railway env add DPI 120       # Bigger text

TIMEZONE & LANGUAGE
    railway env add TZ America/New_York
    railway env add LANGUAGE en_US.UTF-8

⚡ PERFORMANCE TIPS
================================================================================

For Smooth Browsing:
  ✓ Use 1920x1080 resolution (good balance)
  ✓ Close unused tabs (each uses memory)
  ✓ Keep internet speed >1 Mbps
  ✓ Use same region as Railway deployment

For Slow Connections:
  ✓ Reduce to 1280x720 resolution
  ✓ Reduce DPI (72 instead of 96)
  ✓ Close extra browser tabs
  ✓ Disable auto-play videos

For Video Streaming:
  ✓ 720p needs 2-3 Mbps
  ✓ 1080p needs 5+ Mbps
  ✓ Quality adjusts automatically
